# 蓝银草KMZ航线导入指南

一个简洁的静态网页，提供DJI无人机KMZ航线文件导入的完整操作指南。

## 🚁 功能特点
- 支持所有DJI遥控器类型（带屏/不带屏）
- 涵盖Android和iOS手机遥控器
- 图文并茂的操作步骤
- 航点限制解决方案
- 响应式设计，支持移动端

## 📱 支持的设备
- **带屏遥控器**：DJI RC、DJI RC 2、DJI RC Pro
- **安卓手机**：DJI RC N2、DJI RC N3
- **苹果手机**：DJI RC N2、DJI RC N3

## 🚀 使用方法
1. 访问网页：[1978102.xyz](http://1978102.xyz)
2. 选择对应的遥控器类型
3. 按照图文步骤操作
4. 使用航点拆分功能解决航点限制

## 🛠️ 技术栈
- HTML5 + CSS3
- Tailwind CSS（CDN）
- Font Awesome图标
- 纯静态网页，无需服务器
